int eh_par ( long long  num)

Função: eh_par Uso: x = eh_par(num); Verifica se um número é par analisando o resto desse por dois. Caso o resto seja 0 sabe-se que ele é par, caso contrário, ele é impar.

Parâmetros
num: (long long) número que vai ter paridade testada
Retorna
retorno: (int) 1 se for par. 0 se não for par.

Parâmetros
num: (long long) Essa função recebe um numero e retorna (1) se o número pode ser escrito da forma x! em que x é um inteiro menor ou igual
Retorna
retorno: (int) 1 se o número for um fatorial ou (0) se não pode ser escrito da forma x!.
